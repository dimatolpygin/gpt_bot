В файле src/index.js добавь в самое начало (первые 2 строки, до любых импортов):

import { setDefaultResultOrder } from 'node:dns';
setDefaultResultOrder('ipv4first');

Это фикс известного бага Node.js 18+ где DNS резолвит IPv6 вместо IPv4
и Telegram API становится недоступен.

ПРОБЛЕМА 2 — Redis (Docker недоступен)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Используем Upstash — бесплатный облачный Redis, работает с ioredis без изменений в коде.

Шаги для меня (ты только подожди):
1. Я иду на https://console.upstash.com
2. Sign up (бесплатно) → Create Database
3. Region: EU-West или US-East (что ближе к серверу)
4. В настройках базы нахожу вкладку "Details"
5. Копирую строку "Redis URL" формата:
   rediss://default:<password>@<host>.upstash.io:<port>

Как только я дам тебе эту строку — ты обновишь REDIS_URL в файле .env.
